Installation
===
1. virtualenv --no-site-package pymusic-wave
2. source pymusic-wave/bin/activate
3. pip install bottle
4. git clone git://github.com/thinker/pymusic-wave.git
5. cd pymusic-wave
6. python index.py

Dependencies
===
- PyWaveform (https://github.com/superjoe30/PyWaveform)
  - imagemagick
  - mpg123
  - libsndfile
